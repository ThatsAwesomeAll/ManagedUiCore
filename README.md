# ManagedUiCore
Code for a managed UI System

# Usage

* Setup as a plain code based .git submodule
* If you want a proper package with clean dependencies and examples go to ManagedUiPackage

# Steps to enable
* Create git submodule in any folder in you Unity Assets
  * ```git submodule add https://github.com/ThatsAwesomeAll/ManagedUiCore.git $YOURFOLDER$```
  * push, pull and do whatever you can with git
  * DONT push Assets. Only push code
